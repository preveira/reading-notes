# SQL - Prep

### SQL Summary:
Relational databases are a type of database that store data in tables, which are structured into rows and columns. Each table represents a specific entity, such as customers, orders, or products, and each row in the table represents a unique instance of that entity. Columns represent the attributes or properties of the entity.
